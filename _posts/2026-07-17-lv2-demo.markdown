이번 시리즈는 라즈베리파이5(SPI 마스터)가 STM32 Nucleo 보드(SPI 슬레이브, FOC 모터 제어
펌웨어가 돌고 있음)에서 실시간 텔레메트리를 커스텀 SPI DMA 드라이버로 받아오는 이야기다.
글로 들어가기 전에, 이게 실제로 물리적으로 이어져 있다는 걸 영상 두 개로 먼저 보여준다.

위는 Nucleo에 USB 시리얼로 붙어서 SimpleFOC `Commander` 인터페이스로 목표각을 직접
입력하는 화면(`A<라디안값>` — 임피던스 제어라 목표각으로 스프링처럼 끌려간다), 아래는
그 순간 RPi5가 SPI DMA로 받아서 mmap+웹소켓으로 띄운 대시보드다. 둘 다 같은 시점을 찍은
거라, 아래 버튼으로 동시에 재생하면 "커맨드 입력 → 모터가 실제로 움직임 → 그 값이 커널
드라이버를 거쳐 브라우저 차트에 실시간으로 뜸"까지 한 번에 보인다.

<script>
(function () {
  var v1 = document.getElementById('video-motor');
  var v2 = document.getElementById('video-dashboard');
  var btn = document.getElementById('play-both-btn');
  var videos = [v1, v2];

  v1.src = '{{ "/assets/videos/motor-demo.mp4" | relative_url }}';
  v2.src = '{{ "/assets/videos/dashboard-demo.mp4" | relative_url }}';

  function playBoth() {
    videos.forEach(function (v) { v.currentTime = 0; });
    Promise.all(videos.map(function (v) { return v.play(); })).catch(function () {});
  }

  btn.addEventListener('click', playBoth);

  videos.forEach(function (v) {
    v.addEventListener('ended', function () {
      if (videos.every(function (vv) { return vv.ended; })) {
        playBoth();
      }
    });
  });
})();
</script>

두 영상 다 55초 남짓으로 길이를 맞춰뒀다 — "같이 재생" 누르면 같이 시작하고, 둘 다 끝나면
자동으로 같이 처음부터 다시 돈다.
